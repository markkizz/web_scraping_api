Web-Scraping API
======

web scraping using flask

Install
-------

Create a virtualenv and activate it::
```
$ python3 -m venv venv
$ source ./venv/bin/activate
```

Install dependencies
```
$ pip install -e .
```

Run
---
```
$ export FLASK_APP=flaskr
$ export FLASK_ENV=development
$ flask init-db
$ flask run
```
